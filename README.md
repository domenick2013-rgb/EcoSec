# Аналітичний вебсайт (EcoSec Project)

Це персональний статичний вебсайт, розроблений в рамках дисципліни **«Інформаційні технології в забезпеченні економічної безпеки держави»**. 

## 📌 Про проект

Сайт складається з трьох основних сторінок, між якими реалізована зручна навігація:
1. **Головна сторінка (`index.html`)** — загальна інформація про студента (ПІБ, факультет, група, дисципліна).
2. **Резюме (`resume.html`)** — детальне CV з описом професійного досвіду в сфері кібербезпеки (SIEM, XDR/EDR, Cisco Firepower), технічних навичок та освіти.
3. **Аналітичний звіт (`report.html`)** — результати дослідження обраної країни протягом семестру (5 основних загроз та 5 рекомендацій).

## 🚀 Технології
- **HTML5** — структурована та семантична розмітка сторінок.
- **CSS3 (Custom Properties, Flexbox, Animations)** — сучасний стильовий напрямок з використанням ефекту Glassmorphism.
- **Vanilla JavaScript** — інтерактивні елементи, такі як плавна поява контенту при прокручуванні (`Intersection Observer`), зміна фону при русі мишею та автоматична підсвітка активного меню.

## 🎨 Особливості дизайну та функціонал
- **Dark Mode / Light Mode:** Підтримка темної та світлої теми з автоматичним збереженням вибору користувача (`localStorage`).
- **Адаптивність (Responsive Design):** Інтерфейс підлаштовується під будь-які екрани. Додано мобільне меню ("гамбургер").
- **FontAwesome Icons:** Використання векторних іконок для візуального покращення карток та навігації.
- **Завантаження CV:** Додано стильну кнопку для завантаження резюме у форматі PDF.
- **Преміум-вигляд (Glassmorphism):** Напівпрозорі картки з розмиттям фону та м'якими тінями (`box-shadow`), що створюють ефект "об'ємності".
- **Мікроінтеракції:** Плавні CSS-переходи (`transition`) при наведенні на посилання та картки.
- **SEO Оптимізація:** Налаштовано правильні `meta` теги та `favicon`.
